Courses

These courses are personally crafted and moderated by me, Charlie.

Each course will have a free demo then afterwards will be payed for each lesson. Prices will vary depending on the course. Courses will be one on one meetings held at each lesson. Each lesson is active twice a week, and if you cannot make it to a lesson you can receive the online version.

If you would like an extention on a course, you can request any number of weeks at an added constant of $10 per week added.

If you see a course you would like that isn't listed there is a form at the bottom where you can submit a request.
